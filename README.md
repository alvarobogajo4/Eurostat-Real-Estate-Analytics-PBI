# Eurostat-Real-Estate-Analytics-PBI
Dashboard interactivo en Power BI que analiza la asequibilidad de la vivienda en Europa (2015-2024). Aplicación de ETL dinámico y DAX avanzado sobre datos de Eurostat [tipsho60] para identificar riesgos de burbuja inmobiliaria y desajustes entre precios y salarios.
